This plugin adds role-based control of shipping and payment methods for WooCommerce

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory.

2. Activate the plugin  through the 'Plugins' menu in WordPress.

== Usage ==

Visit WooCommerce > Pay Roles Settings and Ship Roles Settings sections in the WooCommerce administration area. Check the desired methods for each payment and shipping method per user role. Activate the Wordpress Groups Plugin (https://wordpress.org/plugins/groups/) to control available payment gateways and shipping methods on a per-group basis.
