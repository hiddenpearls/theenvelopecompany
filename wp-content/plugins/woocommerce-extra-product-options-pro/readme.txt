This plugin helps you to add extra product options in your product page.

== Description ==

= Introduction =
<strong>WooCommerce Extra Product Options</strong> plugin provides an easy way to manage extra product options displayed in product page. This plugin provides an easy to use interface where you can add, edit and delete extra product options. This plugin requires WooCommerce to be installed, activated, and configured.

= BASIC FEATURES =
* <strong>Add extra product option(s).</strong> You can add extra product option(s), these options will be displayed in product page. Add new option form contains options to provide values for Type, Name, Label, Placeholder, Class, Label Class etc. It also has the option to choose validations rules from a predefined list.
* <strong>Edit extra product option(s).</strong> You can edit extra product option(s) properties. In edit form you have the option to provide new values for Name, Type, Label, Placeholder, options, Class, Label Class, validation rules etc(availability of these options may change based on the field types). 
* Available field types are <strong>Input Text</strong> and <strong>Select</strong>
* <strong>Remove extra product option(s).</strong> You can remove extra product option(s) from displaying in product page.
* <strong>Enable/Disable extra product option(s).</strong> You can enable/disable extra product option(s)(temporarily remove) from displaying in product page.
* <strong>Change display order for extra product option(s).</strong> You can easily manage the display order of extra product option(s) from admin side by just moving fields up and down.
* <strong>Reset all the changes.</strong> You can reset all your changes back to the original settings using the button ‘Reset to default options’.

== Installation ==
= Minimum Requirements =
* WooCommerce 2.2 +
* WordPress 3.8 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic installation =
1. Log in to your WordPress dashboard
2. Navigate to the Plugins menu and click Add New.
3. Search and locate 'Woo Extra Product Options' plugin.
4. Click 'Install Now'

= Manual installation =
The manual installation method involves downloading 'Woo Extra Product Options' plugin and uploading it to your webserver via FTP application. You may find more instructions on how to do this here (http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
